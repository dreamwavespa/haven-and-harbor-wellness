# Haven & Harbor — Community Mental Wellness Center

A responsive Next.js 15 website (and companion-app design concept) for a community mental wellness center. Built with TypeScript, Tailwind CSS, and shadcn/ui.

## Pages

- `/` — Home: hero, quick actions, service preview, testimonials
- `/services` — Discover Services: searchable/filterable service catalog
- `/book` — Book an Appointment: 5-step booking wizard
- `/goals` — Track Wellness Goals: progress bars, streaks, mood check-in
- `/resources` — Self-Guided Resources: searchable meditation/article/exercise library
- `/app` — Companion mobile app showcase (phone mockups)

## Getting started

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Deploy

Deploys cleanly to Vercel — just import this repo, framework preset `Next.js`.
